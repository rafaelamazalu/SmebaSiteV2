Accesibilidad Web

La accesibilidad web es la práctica inclusiva de garantizar la accesibilidad a los sitios web, y que las herramientas y las tecnologías estén diseñados y desarrollados para que sean utilizados por el máximo número de personas posibles. Es decir, las personas pueden percibir, comprender, navegar e interactuar con la web sin importar sus limitaciones físicas, cognitivas o tecnológicas.

Evaluación de la accesibilidad Web

La evaluación de la Accesibilidad Web abarca una amplia gama de disciplinasy habilidades y requiere no solo conocimiento sobre aspectos técnicosrelacionados con las tecnologías Web, directrices, estándares y herramientas deevaluación, sino también aspectos no técnicos como la participación de los usuarios finales en el proceso de evaluación.

SMEBA

Con el objetivo de simplificar el proceso de toma de decisiones y reducir la intervención humana, se propone una solución basada en agentes inteligentes para la evaluación de barreras de accesibilidad de acuerdo a las necesidades del usuario, focalizando en aquellos con limitaciones visuales. Así, se presentamos a SMEBA el Sistema Multi-agente para la Evaluación de Barreras de Accesibilidad, cuyo proceso de toma de decisiones se compone de dos etapas:

Una primera instancia de clasificación de las características de un usuario con limitaciones visuales en estereotipos Una segunda instancia de evaluación de adecuación de accesibilidad de los sitios Web que visite, ajustada a las recomendaciones de las Guías de Accesibilidad al Contenido Web 2.1.

Registración

Se solicitará un nombre de usuario y contraseña y, a través de una serie de preguntas relativas a la persona como edad, nombre y tipos de discapacidades de posee, otras indicaciones con el propósito de obtener información técnica como idioma, uso de tecnología asistiva. Además se realizará una serie de ejercicios adaptados a la web para capturar información de capacidad visual del usuario.
